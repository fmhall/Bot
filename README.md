# Farcaster-py Example

This example starts a Farcaster bot using [farcaster-py](https://github.com/a16z/farcaster-py).

## ✨ Features

- Python
- Farcaster-py

## 💁‍♀️ How to use

- Install packages using `poetry install`
- Start the bot using `poetry run python main.py`
